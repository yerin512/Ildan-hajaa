# 일단하자 (ildanhaja)

ADHD 증상자 혹은 일상 업무에서 집중이 어려운 사람들을 위해, **과업을 단계별 스텝으로 쪼개 목표 달성을 돕는 행동 유도 타이머**.

- “지금 뭐 해야 하지?”를 앱이 대신 결정해주고  
- 링 타이머로 시간 감각을 보조하며  
- 매 스텝에서 동일한 액션(했어요 / 30초 더)로 **다음 행동(Next Action)** 을 촉발합니다.

- 예린

## 목표
1. 사용자가 **약속/외출에서 제 시간 출발 성공률을 유의미하게 높인다.**
2. “지금 뭐 해야 하지?”를 앱이 대신 결정해줘서 **행동(Next Action)** 을 촉발한다.
3. 준비물/약속 관련 실수를 줄여 **출발 직전 스트레스를 감소**시킨다.

---

## 서비스 페르소나
- **시간 감각이 약한 ADHD(또는 유사 특성) 사용자**

---

## 핵심 컨셉
- **행동 템플릿**: 외출 준비/독서/운동처럼 자주 하는 행동을 “스텝”으로 쪼개 미리 구성
- **링 타이머**: 남은 시간을 직관적으로 보여주는 원형(링) 카운트다운
- **즉시 행동 버튼**: 매 스텝에서 동일한 버튼(했어요 / 30초 더)로 다음 행동을 유도
- **세션 기록**: 타이머 종료 후 “성공 여부”를 기록

---

## MVP (V1) 요구사항

### 1) 홈(Home)

#### 행동 템플릿 선택 영역 (일단 2개)
- 페이지 상단에 **드롭다운**으로 노출
- 기본 노출 템플릿
  - **외출 준비**
  - **독서**

#### 행동 템플릿
- **스텝 프리뷰** 제공
- 사용자 커스터마이징 가능
  - 스텝 **소요 시간**
  - 스텝 **순서 정렬**
  - 스텝 **이름**
  - **드래그 앤 드롭**으로 순서 수정 가능

#### 기본 템플릿 제공
- **외출 준비 템플릿**
  1. 씻기 (20 mins)
  2. 외모 정돈 (20 mins)
  3. 옷 갈아입기 (5 mins)
  4. 이동 (30 mins)

- **독서 템플릿 (인터벌)**
  1. 읽고 싶은 책 고르기 (10 mins)
  2. 편안한 자세잡고 독서 시작 (25 mins)
  3. 쉬기 (5 mins)
  4. 위 반복

- **운동 템플릿**
  1. 운동복 갈아입기 (10 mins)
  2. 준비 운동하기 (25 mins)
  3. 첫번째 운동 시작 (20 mins)
  4. 쉬기 (5 mins)
  5. 위 반복

---

### 2) 타이머(Timer)

#### 링 타이머
- 링 형태로 **남은 시간 카운트다운**
- 남은 시간이 **3분 미만**이 되면 별도 인터랙션 제공 (예: 화면이 반짝이는 등)

#### 버튼
- 하단에 스텝마다 **동일한 버튼** 존재
  - `했어요`
  - `30초 더`

#### 스텝 진행
- 정해진 스텝이 **순서대로 진행**
- 각 행동 템플릿별 스텝 **최대 10개**

---

### 3) 세션 기록(Session Record)
- **행동 성공 여부 기록**
- 타이머 완료 후, 세션 기록 페이지에서 성공 여부를 기록

---

## 화면 구성(권장 IA)
- 홈
  - 템플릿 드롭다운
  - 스텝 프리뷰 + 편집(시간/이름/순서)
  - 시작 버튼
- 타이머
  - 링 타이머
  - 현재 스텝 정보
  - CTA: 했어요 / 30초 더
- 세션 기록
  - 템플릿/세션 정보
  - 성공/실패(또는 완료/미완료) 기록

## 데이터 모델(초안)
> 구현 전 합의용 스케치

- Template
  - id
  - name (외출 준비/독서/운동 등)
  - steps: Step[]
  - type (normal / interval)

- Step
  - id
  - title
  - durationSec
  - order

- Session
  - id
  - templateId
  - startedAt
  - finishedAt
  - result (success / fail)
  - stepsSnapshot (실행 당시 스텝 스냅샷; 선택)
